## Run Locally

Clone this project

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn
```

Set envs

create a .env with the required envs as shown in `.env.local` with the values.

Start the server

```bash
  yarn dev
```

Go to <http://localhost:3000> on your browser to access the page
